Recipe Finder Application

I combined my Java experience and internship-based app development knowledge to program an android application, "Recipe Finder", with a friend. Together, we used Android Studio and a real-time code sharing plug-in to type through each and every line of code together. Our application takes two user-based inputs: a comma separated list of ingredients and a number of missing ingredients that the user is willing to obtain. The app broadcasts this information to the Spoonacular API — procuring an output of possible recipes onto a scrollable list, arranged from least to most by the number of missing ingredients the user will need to obtain per recipe. The scrollable list allows users to select their recipe of choice and have a full ingredients list and recipe on display through further communication with the API.
